    Exerc√≠cio 4: MLP

    Objetivo

    Implementar uma vers√£o mais profunda do Multi-Layer Perceptron (MLP) com pelo menos 2 camadas escondidas, reutilizando e expandindo o c√≥digo do Exerc√≠cio 3. Este exerc√≠cio demonstra como redes neurais mais profundas podem capturar representa√ß√µes mais complexas e hier√°rquicas dos dados, comparando seu desempenho com arquiteturas mais simples.

    Especifica√ß√µes T√©cnicas:

    • Dataset: Mesmo do Exerc√≠cio 3 (1500 amostras, 4 features, 3 classes)
    • Arquitetura: 4 ‚Üí 32 ‚Üí 16 ‚Üí 3 neur√¥nios (2 camadas escondidas)
    • Fun√ß√µes de Ativa√ß√£o: tanh (camadas ocultas) + softmax (sa√≠da)
    • Fun√ß√£o de Perda: Categorical Cross-Entropy
    • Otimizador: Gradient Descent
    • Learning Rate: 0.05 (reduzido para maior estabilidade)
    • √âpocas: 400 (mais treinamento devido √† complexidade)

    1. Motiva√ß√£o para Redes Mais Profundas

    Redes neurais profundas oferecem v√°rias vantagens sobre arquiteturas rasas:

    Capacidade de Representa√ß√£o Hier√°rquica:

    • Primeira camada escondida (32 neur√¥nios): captura caracter√≠sticas b√°sicas
    • Segunda camada escondida (16 neur√¥nios): combina caracter√≠sticas em padr√µes complexos
    • Camada de sa√≠da (3 neur√¥nios): classifica√ß√£o final baseada em representa√ß√µes hier√°rquicas

    Maior Expressividade:

    • Mais par√¢metros permitem modelar rela√ß√µes n√£o-lineares mais complexas
    • Capacidade de aproximar fun√ß√µes mais sofisticadas
    • Melhor separa√ß√£o de classes com fronteiras de decis√£o mais elaboradas

    2. Implementa√ß√£o da Classe MLP Profundo

    2.1 Estrutura Principal e Inicializa√ß√£o Din√¢mica

    A nova classe DeepMultiClassMLP √© projetada para suportar um n√∫mero arbitr√°rio de camadas escondidas, usando um sistema din√¢mico de inicializa√ß√£o de par√¢metros.

    O que este c√≥digo faz:

    • Aceita uma lista de tamanhos para camadas escondidas (hidden_sizes)
    • Inicializa dinamicamente pesos e bias para cada camada
    • Utiliza Xavier/Glorot initialization para todas as camadas
    • Armazena par√¢metros em dicion√°rios indexados por camada
    class DeepMultiClassMLP:

    def __init__(self, input_size, hidden_sizes, num_classes, learning_rate=0.05):
        self.input_size = input_size
        self.hidden_sizes = hidden_sizes
        self.num_classes = num_classes
        self.learning_rate = learning_rate
        self.num_layers = len(hidden_sizes) + 1  

        self.weights = {}
        self.biases = {}

        prev_size = input_size
        for i, hidden_size in enumerate(hidden_sizes):
            self.weights[f'W{i+1}'] = np.random.randn(hidden_size, prev_size) * np.sqrt(2.0 / prev_size)
            self.biases[f'b{i+1}'] = np.zeros((hidden_size, 1))
            prev_size = hidden_size

        self.weights[f'W{self.num_layers}'] = np.random.randn(num_classes, prev_size) * np.sqrt(2.0 / prev_size)
        self.biases[f'b{self.num_layers}'] = np.zeros((num_classes, 1))

        self.loss_history = []
        self.accuracy_history = []

    2.2 Fun√ß√µes de Ativa√ß√£o Reutilizadas

    As fun√ß√µes de ativa√ß√£o permanecem as mesmas, mas agora s√£o aplicadas a m√∫ltiplas camadas escondidas. A combina√ß√£o tanh + softmax continua sendo eficaz para classifica√ß√£o multiclasse profunda.

    O que este c√≥digo faz:

    • Mant√©m tanh para todas as camadas escondidas (preserva gradientes)
    • Preserva softmax para normaliza√ß√£o probabil√≠stica multiclasse
    • Utiliza one-hot encoding para labels categ√≥ricos
    def tanh(self, z):
    return np.tanh(z)

def tanh_derivative(self, z):
    return 1 - np.tanh(z)**2

def softmax(self, z):
    z_shifted = z - np.max(z, axis=0, keepdims=True)
    exp_z = np.exp(z_shifted)
    return exp_z / np.sum(exp_z, axis=0, keepdims=True)

def one_hot_encode(self, y, num_classes):
    one_hot = np.zeros((num_classes, len(y)))
    one_hot[y.astype(int), np.arange(len(y))] = 1
    return one_hot

    3. Forward Pass Profundo

    3.1 Propaga√ß√£o Atrav√©s de M√∫ltiplas Camadas

    O forward pass generalizado percorre dinamicamente todas as camadas escondidas, aplicando transforma√ß√µes lineares seguidas de ativa√ß√µes tanh, antes da classifica√ß√£o final com softmax.

    O que este c√≥digo faz:

    • Itera atrav√©s de todas as camadas escondidas automaticamente
    • Aplica tanh em cada camada escondida para n√£o-linearidade
    • Armazena todos os valores intermedi√°rios para backpropagation
    • Termina com softmax para distribui√ß√£o de probabilidades multiclasse
    def deep_forward(self, X):
    if X.ndim == 1:
        X = X.reshape(1, -1)

    cache = {}
    A = X.T
    cache['A0'] = A

    for layer in range(1, self.num_layers):
        Z = self.weights[f'W{layer}'] @ A + self.biases[f'b{layer}']
        A = self.tanh(Z)
        cache[f'Z{layer}'] = Z
        cache[f'A{layer}'] = A

    Z_out = self.weights[f'W{self.num_layers}'] @ A + self.biases[f'b{self.num_layers}']
    A_out = self.softmax(Z_out)
    cache[f'Z{self.num_layers}'] = Z_out
    cache[f'A{self.num_layers}'] = A_out

    return A_out, cache

def deep_predict_proba(self, X):
    output, _ = self.forward(X)
    return output.T

def deep_predict(self, X):
    probabilities = self.predict_proba(X)
    return np.argmax(probabilities, axis=1)

DeepMultiClassMLP.forward = deep_forward
DeepMultiClassMLP.predict_proba = deep_predict_proba
DeepMultiClassMLP.predict = deep_predict

    4. Backward Pass Profundo

    4.1 Backpropagation Atrav√©s de M√∫ltiplas Camadas

    O algoritmo de backpropagation √© generalizado para funcionar com qualquer n√∫mero de camadas escondidas, propagando gradientes de volta atrav√©s de toda a arquitetura profunda.

    O que este c√≥digo faz:

    • Calcula gradientes come√ßando pela camada de sa√≠da
    • Propaga erros de volta atrav√©s de todas as camadas escondidas
    • Utiliza a regra da cadeia para camadas m√∫ltiplas
    • Armazena gradientes para todas as camadas em um dicion√°rio
    def deep_compute_loss(self, y_true, y_pred):
    if y_true.ndim == 1:
        y_true_onehot = self.one_hot_encode(y_true, self.num_classes)
    else:
        y_true_onehot = y_true

    epsilon = 1e-15
    y_pred_clipped = np.clip(y_pred, epsilon, 1 - epsilon)
    loss = -np.mean(np.sum(y_true_onehot * np.log(y_pred_clipped), axis=0))
    return loss

def deep_backward(self, X, y, cache):
    m = X.shape[0]
    gradients = {}

    y_onehot = self.one_hot_encode(y, self.num_classes)

    A_out = cache[f'A{self.num_layers}']
    dZ = A_out - y_onehot

    for layer in range(self.num_layers, 0, -1):
        A_prev = cache[f'A{layer-1}']

        gradients[f'dW{layer}'] = (1/m) * np.dot(dZ, A_prev.T)
        gradients[f'db{layer}'] = (1/m) * np.sum(dZ, axis=1, keepdims=True)

        if layer > 1:
            dA = np.dot(self.weights[f'W{layer}'].T, dZ)
            Z_prev = cache[f'Z{layer-1}']
            dZ = dA * self.tanh_derivative(Z_prev)

    return gradients

DeepMultiClassMLP.compute_loss = deep_compute_loss
DeepMultiClassMLP.backward = deep_backward

    5. Treinamento da Rede Profunda

    5.1 Loop de Treinamento Adaptado

    O m√©todo de treinamento √© adaptado para lidar com a maior complexidade da rede profunda, incluindo atualiza√ß√µes din√¢micas de par√¢metros para todas as camadas.

    O que este c√≥digo faz:

    • Executa mais √©pocas devido √† maior complexidade da rede
    • Atualiza dinamicamente todos os pesos e bias de todas as camadas
    • Monitora converg√™ncia com maior frequ√™ncia
    • Garante estabilidade durante o treinamento profundo
    def deep_fit(self, X_train, y_train, epochs=200, print_every=50):
    self.loss_history = []
    self.accuracy_history = []

    print(f"Treinando rede profunda por {epochs} √©pocas...")

    for epoch in range(epochs):
        # forward pass
        output, cache = self.forward(X_train)

        # perda
        loss = self.compute_loss(y_train, output)
        self.loss_history.append(loss)

        #  acur√°cia
        predictions = np.argmax(output, axis=0)
        accuracy = np.mean(predictions == y_train)
        self.accuracy_history.append(accuracy)

        # backward pass
        gradients = self.backward(X_train, y_train, cache)

        # Atualiza
        for layer in range(1, self.num_layers + 1):
            self.weights[f'W{layer}'] -= self.learning_rate * gradients[f'dW{layer}']
            self.biases[f'b{layer}'] -= self.learning_rate * gradients[f'db{layer}']

        if (epoch + 1) % print_every == 0:
            print(f"√âpoca {epoch+1}/{epochs} - Loss: {loss:.4f} - Acc: {accuracy:.4f}")

DeepMultiClassMLP.fit = deep_fit

    def plot_deep_decision_boundary(model, X, y, title="Fronteiras de Decis√£o"):

    plt.xlabel('Feature 1 (normalizada)')
    plt.ylabel('Feature 2 (normalizada)')
    plt.title(title)
