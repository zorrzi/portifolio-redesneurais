    Exerc√≠cio 3: MLP Multiclasse

    Objetivo

    Implementar um Multi-Layer Perceptron (MLP) do zero para resolver um problema de classifica√ß√£o multiclasse (3 classes), utilizando apenas a biblioteca NumPy para c√°lculos matem√°ticos.     Especifica√ß√µes T√©cnicas:

    3. Implementa√ß√£o da Classe MLP Multiclasse

    3.1 Estrutura Principal e Inicializa√ß√£o

    Vamos criar uma classe espec√≠fica para classifica√ß√£o multiclasse que encapsula toda a funcionalidade do nosso MLP. A inicializa√ß√£o √© adaptada para lidar com m√∫ltiplas classes de sa√≠da, utilizando inicializa√ß√£o Xavier para melhor converg√™ncia.

    O que este c√≥digo faz:

    • Inicializa pesos usando Xavier/Glorot initialization
    • Configura arquitetura 4‚Üí16‚Üí3 para o problema multiclasse
    • Prepara estruturas para armazenar hist√≥rico de treinamento
    class MultiClassMLP:

    def __init__(self, input_size, hidden_size, num_classes, learning_rate=0.1):
        self.input_size = input_size
        self.hidden_size = hidden_size
        self.num_classes = num_classes
        self.learning_rate = learning_rate

        self.W1 = np.random.randn(hidden_size, input_size) * np.sqrt(2.0 / input_size)
        self.b1 = np.zeros((hidden_size, 1))
        self.W2 = np.random.randn(num_classes, hidden_size) * np.sqrt(2.0 / hidden_size)
        self.b2 = np.zeros((num_classes, 1))

        self.loss_history = []
        self.accuracy_history = []

    3.2 Fun√ß√µes de Ativa√ß√£o para Classifica√ß√£o Multiclasse

    As fun√ß√µes de ativa√ß√£o s√£o adaptadas para classifica√ß√£o multiclasse. Utilizamos tanh para a camada oculta e softmax para a sa√≠da, que produz probabilidades normalizadas para cada classe.

    O que este c√≥digo faz:

    • Implementa tanh e sua derivada para camada oculta
    • Implementa softmax para sa√≠da multiclasse com estabilidade num√©rica
    • Implementa one-hot encoding para converter labels categ√≥ricos
    def tanh(self, z):
    return np.tanh(z)

def tanh_derivative(self, z):
    return 1 - np.tanh(z)**2

def softmax(self, z):
    z_shifted = z - np.max(z, axis=0, keepdims=True)
    exp_z = np.exp(z_shifted)
    return exp_z / np.sum(exp_z, axis=0, keepdims=True)

def one_hot_encode(self, y, num_classes):
    one_hot = np.zeros((num_classes, len(y)))
    one_hot[y.astype(int), np.arange(len(y))] = 1
    return one_hot

MultiClassMLP.tanh = tanh
MultiClassMLP.tanh_derivative = tanh_derivative
MultiClassMLP.softmax = softmax
MultiClassMLP.one_hot_encode = one_hot_encode

    3.3 Forward Pass para Classifica√ß√£o Multiclasse

    O forward pass processa os dados atrav√©s da rede at√© produzir probabilidades para cada uma das tr√™s classes. O softmax garante que as probabilidades somem 1.

    O que este c√≥digo faz:

    • Propaga dados atrav√©s das camadas da rede
    • Aplica transforma√ß√µes lineares e n√£o-lineares
    • Produz distribui√ß√£o de probabilidades sobre as classes
    • Armazena valores intermedi√°rios para backpropagation
    def forward(self, X):
    if X.ndim == 1:
        X = X.reshape(1, -1)
    A0 = X.T

    # Camada oculta: linear + tanh
    Z1 = self.W1 @ A0 + self.b1
    A1 = self.tanh(Z1)

    # Camada de sa√≠da: linear + softmax
    Z2 = self.W2 @ A1 + self.b2
    A2 = self.softmax(Z2)

    cache = {'A0': A0, 'Z1': Z1, 'A1': A1, 'Z2': Z2, 'A2': A2}
    return A2, cache

def predict_proba(self, X):
    output, _ = self.forward(X)
    return output.T

def predict(self, X):
    probabilities = self.predict_proba(X)
    return np.argmax(probabilities, axis=1)

MultiClassMLP.forward = forward
MultiClassMLP.predict_proba = predict_proba
MultiClassMLP.predict = predict

    3.4 Fun√ß√£o de Perda e Backward Pass Multiclasse

    A fun√ß√£o de perda Categorical Cross-Entropy √© espec√≠fica para problemas multiclasse. O backward pass calcula gradientes considerando a natureza multiclasse do problema.

    O que este c√≥digo faz:

    • Implementa Categorical Cross-Entropy Loss
    • Calcula gradientes espec√≠ficos para classifica√ß√£o multiclasse
    • Utiliza a propriedade simplificada do gradiente softmax + cross-entropy
    • Propaga erros de volta atrav√©s de todas as camadas
    def compute_loss(self, y_true, y_pred):
    if y_true.ndim == 1:
        y_true_onehot = self.one_hot_encode(y_true, self.num_classes)
    else:
        y_true_onehot = y_true

    epsilon = 1e-15
    y_pred_clipped = np.clip(y_pred, epsilon, 1 - epsilon)

    loss = -np.mean(np.sum(y_true_onehot * np.log(y_pred_clipped), axis=0))
    return loss

def backward(self, X, y, cache):
    m = X.shape[0]
    A0, A1, A2 = cache['A0'], cache['A1'], cache['A2']
    Z1 = cache['Z1']

    y_onehot = self.one_hot_encode(y, self.num_classes)

    dZ2 = A2 - y_onehot

    dW2 = (1/m) * np.dot(dZ2, A1.T)
    db2 = (1/m) * np.sum(dZ2, axis=1, keepdims=True)

    dA1 = np.dot(self.W2.T, dZ2)
    dZ1 = dA1 * self.tanh_derivative(Z1)

    dW1 = (1/m) * np.dot(dZ1, A0.T)
    db1 = (1/m) * np.sum(dZ1, axis=1, keepdims=True)

    return dW1, db1, dW2, db2

MultiClassMLP.compute_loss = compute_loss
MultiClassMLP.backward = backward

    3.5 Loop de Treinamento para Classifica√ß√£o Multiclasse

    O m√©todo de treinamento coordena todo o processo de aprendizagem para o problema multiclasse, monitorando tanto a perda quanto a acur√°cia multiclasse durante o treinamento.

    O que este c√≥digo faz:

    • Executa m√∫ltiplas √©pocas de treinamento
    • Calcula acur√°cia multiclasse (classe com maior probabilidade)
    • Atualiza par√¢metros usando Gradient Descent
    • Monitora converg√™ncia durante o processo
    def fit(self, X_train, y_train, epochs=200, print_every=50):
    self.loss_history = []
    self.accuracy_history = []

    print(f"Treinando por {epochs} √©pocas...")

    for epoch in range(epochs):
        # forward pass
        output, cache = self.forward(X_train)

        #  perda
        loss = self.compute_loss(y_train, output)
        self.loss_history.append(loss)

        #  acur√°cia
        predictions = np.argmax(output, axis=0)
        accuracy = np.mean(predictions == y_train)
        self.accuracy_history.append(accuracy)

        # backward pass
        dW1, db1, dW2, db2 = self.backward(X_train, y_train, cache)

        # atualiza
        self.W1 -= self.learning_rate * dW1
        self.b1 -= self.learning_rate * db1
        self.W2 -= self.learning_rate * dW2
        self.b2 -= self.learning_rate * db2

        if (epoch + 1) % print_every == 0:
            print(f"√âpoca {epoch+1}/{epochs} - Loss: {loss:.4f} - Acc: {accuracy:.4f}")


MultiClassMLP.fit = fit
